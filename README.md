# Note Text Polish

Markdownで書いた文章を、noteへ投稿する前に整形・確認するための小さなWebアプリです。

主に小説や長文の本文を想定しています。Markdownからnoteへ貼り付けると段落が詰まってしまう場合に、一度このアプリへ貼り付けて、読みやすい段落に整えてからコピーできます。

## Features

- Markdown本文の貼り付け
- note投稿向けの段落整形
- 「うさぴこ基準」「標準」「note基準」の読み味調整
- PC表示とスマホ表示の並列プレビュー
- 文字数・段落数の確認
- noteの編集者、スマホ小説家、小説の編集者の視点による簡易チェック
- 整形後テキストのコピー
- 入力内容と設定の自動保存

## Usage

1. `index.html` をブラウザで開きます。
2. 左側の「貼り付け」にMarkdown本文を貼り付けます。
3. 「読み味」で余白の強さを調整します。
4. 「note向け整形」を押します。
5. PC / スマホのプレビューを確認します。
6. 問題なければ「コピー」を押して、noteへ貼り付けます。

## 読み味

### うさぴこ基準

小説の密度と没入感を残す設定です。余白を増やしすぎず、本文の流れを優先します。

### 標準

noteで読み落としにくいよう、句点ごとの余白を標準的に入れます。小説の読みやすさとnote向けの見やすさの中間です。

### note基準

noteらしい余白を強めに出す設定です。長い本文をスマホで追いやすいよう、文のまとまりを大きくほぐします。

## 校正視点

### noteの編集者

スマホ画面で長く見える段落や、見出しまわりの余白を確認します。

### スマホ小説家

画面内で息継ぎしやすいか、会話文の前後で視線が戻りやすいかを確認します。

### 小説の編集者

長い台詞や、文末リズムの単調さを確認します。

## Files

- `index.html`: アプリ画面
- `styles.css`: レイアウトと見た目
- `script.js`: 整形、プレビュー、校正チェックの処理

## Notes

このアプリはすべてブラウザ上で動作します。サーバーや外部APIは使いません。

入力した文章はブラウザのローカルストレージに保存されます。共有PCなどで使う場合は、使用後に「消去」を押してください。
